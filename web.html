HTML Cheat Sheet

Basics Tags
<html></html>

Erzeugt ein HTML Dokument.

<head></head>

Nicht sichtbare Informationen im Dokument, z.B. Seitentitel

<body></body>

Sichtbarer Teil des Dokuments

<title></title>

Seitentitle im Browserfenster

<html>
    <head>
        <title>Seitentitle</title>
    </head>
    <body>
        Seiteninhalt
    </body>
</html>

Links
<a href="URL"></a>

Erzeugt einen Hyperlink zu einer URL oder HTML-Dokument.

<a href="mailto:EMAIL_ADDRESS"></a>

Erzeugt einen Hyperlink zu einer EMail-Adresse (Standard Mail-Programm)

<a name="NAME"></a>

Erzeugt einen Anker innerhalb eines Dokuments.

<a href="#NAME"></a>

Erzeugt einen Hyperlink auf einen Anker (auch für ein anderes Dokument möglich).

Formulare
<form></form>
Definiert ein Formular.
<select name=?></select>
Erzeugt ein Pulldown Menü.
<select multiple name=? size=?></select>
Auswahlliste mit Mehrfachauswahl. Size gbit die Anzahl der sichtbaren Einträge aus.
<option></option>
Einträge für ein Pulldown Menü bzw. eine Auswahlliste.
<textarea name=? cols=? rows=?></textarea>
Textbox; Cols definiert die Anzahl der Zeichen in einer Zeile; Rows die Anzahl der Reihen.
<input type="checkbox" name=? value=?>
Erzeugt eine Checkbox. Wenn aktiv, wird der Wert (value) übertragen.
Alternativ type="radio".
<input type="text" name=? size=?>
Einzeiliges Eingabefeld. Size entspricht der Anzahl an Zeichen..
<input type="submit" value=?>
Button zum absenden des Formulars. 
<form>
    <select name="Auswahl">
        <option>Eintrag 1</option>
        <option>Eintrag 3</option>
    </select>
    <input type="checkbox" name="Check">
    <input type="submit" value="submit">
</form>
